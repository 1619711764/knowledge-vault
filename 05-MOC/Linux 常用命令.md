# Linux 常用命令 — MOC

> 从零掌握 Linux 命令行

## 命令分类

### 基础操作

- [[Linux 文件操作]] — ls、cd、cp、mv、rm、mkdir、find
- [[Linux 文本处理]] — cat、grep、head、tail、wc、sort、uniq
- [[Linux 系统管理]] — ps、top、df、du、free、kill、systemctl
- [[Linux 权限管理]] — chmod、chown、sudo、umask

### 进阶

- [[Linux 管道与重定向]] — `|`、`>`、`>>`、`<`
- [[Linux 网络命令]] — curl、wget、ping、ss、nc
- [[Linux 压缩打包]] — tar、gzip、zip
- [[Linux 进程与后台]] — &、nohup、jobs、fg、bg

## 学习技巧

1. 不要死记 — 用 `man 命令` 或 `命令 --help` 随时查
2. 装一个 Linux 虚拟机（或 WSL）上手敲
3. 每次学一个新命令，在这里写一条 [[卡片笔记法 Zettelkasten|卡片笔记]]

## Shell 环境

当前使用的 shell 是 **bash**（`echo $SHELL` 查看）。在 Windows 上也可以用 WSL 或 Git Bash。
