# 欢迎来到你的知识库

## 快速上手

1. **看一圈** — 左侧文件树浏览已创建的示例笔记
2. **图谱视图** — 左侧菜单 → "图表"，查看 [[个人知识管理]] 的节点网络
3. **双向链接** — 打开任意笔记（如 [[Obsidian]]），查看右侧 "反向链接" 面板
4. **搜索** — `Ctrl+Shift+F` 全文搜索关键词
5. **试试创建** — `Ctrl+N` 新建笔记，用 `[[` 添加链接

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建笔记 |
| `Ctrl+O` | 快速打开 |
| `Ctrl+P` | 命令面板（万能入口） |
| `Ctrl+E` | 切换编辑/预览模式 |
| `Ctrl+Shift+F` | 全文搜索 |
| `Ctrl+Shift+D` | 创建日记 |

## 推荐第一步

打开 [[个人知识管理]] — 这是你的 MOC 入口，从这里出发探索。

然后写你的第一条 [[卡片笔记法 Zettelkasten|卡片笔记]]。

## 核心方法

这套系统基于 **[[卡片笔记法 Zettelkasten]]** 和 **[[第二大脑]]** 理念搭建，用 **[[双向链接]]** 取代传统文件夹层级，最终在 **[[知识图谱]]** 中可视化你的思维网络。
