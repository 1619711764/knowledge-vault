# 第二大脑

> 人类大脑擅长**产生想法**，不擅长**存储事实**。把记忆外包给工具，把大脑留给思考。

## 概念的起源

Tiago Forte 在《构建第二大脑》中提出，现代知识工作者的核心问题不是"信息不足"，而是**信息过载但知识贫乏**。

## CODE 工作流

```
Capture  →  Organize  →  Distill  →  Express
 (捕获)      (整理)        (提炼)        (输出)
```

### C — Capture（捕获）
有什么想法、灵感、链接，立刻记下来。不要过滤，不要犹豫。

→ 在本 Vault 中对应：`00-Inbox`

### O — Organize（整理）
定期整理 Inbox，把有价值的内容归档到合适的目录，加上 `[[双向链接]]`。

→ 本 Vault 对应：`01-Notes/` + `05-MOC/`

### D — Distill（提炼）
不是复制粘贴，而是用 [[渐进式总结]] 的方法逐层提炼。

→ 本 Vault 对应：加粗 → 高亮 → 摘要

### E — Express（输出）
写文章、做分享、写代码 — 把知识用起来。知识不被使用就等于不存在。

→ 输出的东西又可以 Capture 回来，形成闭环

## 建立第二大脑的 4 个习惯

1. **任何想法立刻记录** — 不管手机备忘录还是 Obsidian，别等"以后"
2. **定期清空 Inbox** — 每周花 15 分钟整理 `00-Inbox`
3. **写下来再忘掉** — 清空大脑，专注当下
4. **反哺输出** — 写不出东西的时候，翻翻你的第二大脑，答案就在里面

## 第二大脑 ≠ Obsidian

第二大脑是**方法论**，[[Obsidian]] 是**工具**。你也可以用纸质卡片、Notion、Logseq 来实践。但 [[卡片笔记法 Zettelkasten]] + Obsidian 的组合目前公认最有效。

> 你的大脑是 CPU，不是硬盘。让第二大脑当硬盘，你专注运算。
