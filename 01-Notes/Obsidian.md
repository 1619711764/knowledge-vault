# Obsidian

> 本地优先的知识管理工具，核心特色：[[双向链接]] + [[知识图谱]] + Markdown + 1000+ 插件

## 和 Notion/语雀/飞书的区别

| | Notion / 飞书 | Obsidian |
|---|:---:|:---:|
| 数据存储 | 云端（别人服务器） | 本地文件夹 |
| 离线可用 | 部分支持 | 完全可用 |
| 格式 | 私有格式 | 纯文本 `.md` |
| 双向链接 | 弱 | **核心功能** |
| 图谱视图 | 无 | **内置** |
| 迁移风险 | 高（格式锁定） | 低（纯文本） |
| 协作 | 强 | 弱（本就不是为协作设计） |
| 免费 | 有限制 | 个人使用全免费 |

## 界面速览

```
┌─ 左侧边栏 ────┬── 编辑区 ────────────┬── 右侧边栏 ────┐
│ 文件浏览器     │                      │ 反向链接        │
│ 搜索          │  # 标题               │ 标签面板        │
│ 书签          │  正文内容...          │ 大纲           │
│               │  [[链接到其他笔记]]    │               │
│               │                      │               │
└──────────────┴──────────────────────┴────────────────┘
```

## 必知快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 快速打开文件 |
| `Ctrl+P` | 命令面板（万能入口） |
| `Ctrl+N` | 新建笔记 |
| `Ctrl+E` | 编辑/阅读模式切换 |
| `Ctrl+G` | 打开图谱 |
| `Ctrl+Shift+F` | 全文搜索 |
| `Ctrl+Shift+D` | 创建今日日记 |

## 核心插件推荐

| 插件 | 一句话 |
|------|--------|
| **[[Dataview]]** | 把 Vault 当数据库查 |
| **Templater** | 高级模板（比自带模板强） |
| **Calendar** | 日历视图 + 日记入口 |
| **Excalidraw** | 无限画布，自由画图 |
| **Kanban** | 看板任务管理 |
| **Style Settings** | 精细调整主题 |

## 目录结构习惯

```
Vault/
├── 00-Inbox/     ← 快速捕获，不分类
├── 01-Notes/     ← 主要笔记
├── 02-Daily/     ← 日记
├── 03-Templates/ ← 模板
├── 04-Attachments/ ← 图片、附件
└── 05-MOC/       ← 内容地图（索引页）
```

> Obsidian 本身免费，同步用 iCloud/OneDrive/Dropbox，完全不需要付费版。如果你需要跨设备同步且不想用云盘，可以买 Obsidian Sync。
