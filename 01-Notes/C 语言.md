# C 语言

> C 语言是几乎所有现代编程语言的"祖先"。[[Python 学习路线|Python]] 的解释器 CPython 本身就是用 C 写的。

## 核心特点

- **编译型** — 需要编译成二进制才能运行
- **静态类型** — 变量必须声明类型
- **手动内存管理** — `malloc()` / `free()`，没有垃圾回收
- **极高性能** — 接近硬件的效率

## 和 Python 的关键区别

| | C | Python |
|------|---|--------|
| 类型 | 静态：`int x = 5;` | 动态：`x = 5` |
| 编译 | 需要（gcc/clang） | 不需要（解释型） |
| 内存 | 手动管理 | 自动 GC |
| 代码块 | `{}` 花括号 | 缩进 |
| 速度 | 极快 | 较慢（对多数场景够用） |
| 学习曲线 | 陡峭 | 平缓 |

## Hello World

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

编译运行：`gcc hello.c -o hello && ./hello`

> 学习 C 可以帮助你理解 [[Python 基础语法|Python 底层原理]]（内存模型、指针、栈与堆）。但作为初学者，先从 [[Python 学习路线|Python]] 入手更友好。
