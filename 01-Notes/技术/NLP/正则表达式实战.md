# 正则表达式实战

> 正则 = 文本查找和提取的"瑞士军刀"，面试必考，运维/数据分析日常使用频率极高

## 核心符号（只记这些）

| 符号 | 含义 | 示例 |
|------|------|------|
| `.` | 任意字符 | `a.b` 匹配 "acb"、"a b" |
| `*` | 前一个字符 0 次或多次 | `a*` 匹配 ""、"a"、"aaa" |
| `+` | 前一个字符 1 次或多次 | `a+` 匹配 "a"、"aaa" |
| `?` | 前一个字符 0 次或 1 次 | `colou?r` 匹配 "color"、"colour" |
| `\d` | 数字 | 等价 `[0-9]` |
| `\w` | 字母数字下划线 | 等价 `[a-zA-Z0-9_]` |
| `\s` | 空白字符 | 空格、Tab、换行 |
| `[]` | 字符集合 | `[abc]` 匹配 a 或 b 或 c |
| `[^]` | 取反 | `[^0-9]` 非数字 |
| `^` | 行开头 | `^Error` 行首是 Error |
| `$` | 行结尾 | `done$` 行尾是 done |
| `()` | 分组（捕获） | `(\d{3})-(\d{4})` 捕获两组数字 |
| `|` | 或 | `Error|Warn` |

## Python re 模块速查

```python
import re

# search — 找第一个匹配
m = re.search(r'\d{3}-\d{4}-\d{4}', '电话: 139-1234-5678')
print(m.group())  # 139-1234-5678

# findall — 找所有匹配
re.findall(r'\d+', '订单 123 和 456')  # ['123', '456']

# sub — 替换
re.sub(r'\s+', ' ', 'hello   world')   # 'hello world'

# 分组捕获
m = re.search(r'(\w+)@(\w+\.\w+)', '邮箱: test@qq.com')
print(m.group(1))  # test
print(m.group(2))  # qq.com
```

## 三个高频面试题

### 1. 提取日志中的 IP

```python
log = "2026-05-26 14:32:01 ERROR [server-03] timeout: 10.0.1.5:3306"
ip = re.search(r'\d+\.\d+\.\d+\.\d+', log).group()
# "10.0.1.5"
```

### 2. 验证手机号

```python
def is_phone(s):
    return bool(re.match(r'^1[3-9]\d{9}$', s))
```

### 3. 提取日志时间、级别、IP（结构化）

```python
pattern = r'(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) (\w+) .*? (\d+\.\d+\.\d+\.\d+)'
m = re.search(pattern, log)
time, level, ip = m.groups()
```
