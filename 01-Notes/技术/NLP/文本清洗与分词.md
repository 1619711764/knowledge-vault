# 文本清洗与分词

> 文本清洗 = 把"脏数据"变干净；分词 = 把句子切成词

## 文本清洗三步法

### 1. 去除噪音

```python
import re

text = "  这个商品@#真不错！！！五星好评~\n  "
text = text.strip()                              # 去首尾空白
text = re.sub(r'[^一-龥a-zA-Z0-9]', '', text)    # 去特殊字符
text = re.sub(r'\s+', ' ', text)                 # 去多余空格
```

### 2. 统一格式

```python
text = text.replace('“', '"').replace('”', '"')  # 全角引号
text = text.lower()                                         # 英文字母小写
```

### 3. 去停用词

```python
stopwords = {'的', '了', '是', '在', '和', '就', '都', '也', '把'}
words = [w for w in words if w not in stopwords]
```

## jieba 分词

```python
import jieba

list(jieba.cut('我爱北京天安门'))                    # 精确模式
list(jieba.cut('我爱北京天安门', cut_all=True))      # 全模式
list(jieba.cut_for_search('我爱北京天安门'))         # 搜索模式

jieba.add_word('大语言模型')                         # 添加自定义词
list(jieba.cut('大语言模型很好用'))                  # ['大语言模型', '很', '好用']
```

## 完整流程

```python
import re, jieba
from collections import Counter

def process(text):
    text = text.strip()
    text = re.sub(r'[^一-龥a-zA-Z0-9\s]', '', text)
    words = list(jieba.cut(text))
    stopwords = {'的', '了', '是', '在'}
    words = [w for w in words if w.strip() and w not in stopwords]
    return words

# 面试常问：10万条评论找高频词

    all_words.extend(process(comment))
top100 = Counter(all_words).most_common(100)
```

## 面试话术

> 问："jieba 三种模式的区别？"
> 答："精确模式最常用，把句子精确切开；全模式把所有可能的词都列出来，速度快但冗余；搜索模式对长词进一步切分，适合搜索场景。日常用精确模式为主。"
