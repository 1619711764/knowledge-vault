# 卡片笔记法 Zettelkasten

> 由德国社会学家 Niklas Luhmann 发明的笔记方法，他用这个方法在 30 年内出版了 58 本专著

## 核心原则

1. **原子化** — 一篇笔记只记录一个想法
2. **关联性** — 用 [[双向链接]] 把笔记连成网络
3. **用自己的话** — 不复制粘贴，要消化后重写
4. **不分类** — 让笔记之间的联系自然生长，不强加层级

## 三种笔记类型

- **闪念笔记** → 快速捕获，暂存在 `00-Inbox`
- **文献笔记** → 读书看文章时的摘要和理解
- **永久笔记** → 经过整理的、用自己话写的一页纸

## 工作流

1. 随时记录闪念 → [[00-Inbox]]
2. 定期整理 → 转化为永久笔记
3. 添加 [[双向链接]] 连接到已有 [[内容地图 MOC]]
4. 从笔记网络中产生新想法

## 为什么有效

克服了「空白页恐惧症」。你不再需要面对一个空白的主题从头开始 — 把已有卡片排在一起，文章就自然出现了。
